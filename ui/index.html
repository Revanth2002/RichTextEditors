Design Specs: Habitat
Configure Project Design: Quintin, Peter. Last updated 11.30.14 by Peter

Problem:

The interaction model in Organize Content is confusing and not consistent with the rest of Habitat.

Users at publishers need to perform bulk content organization operations. The goal is to improve usability overall, and support publisher use cases for managing many files.

UX Changes to TOC View Center Panel

Multi-select via Command/Control

· You can select individual pages (and blueprint cards).
· Selecting a page that has child pages will select all children as well.
· No support for multi-select of chapters or units.
